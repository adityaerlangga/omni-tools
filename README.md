# Universal Tools

Personal self-hosted toolkit branded **Universal Tools (UT)** with a Nextcloud-inspired UI.

All processing stays on the client — nothing leaves your device.

## Theme

| Token | Value |
| --- | --- |
| Primary blue | `#0082C9` |
| Accent | `#1CAFFF` |
| Typography | Inter |

Not affiliated with Nextcloud GmbH.

## Development

```bash
npm install
npm run dev
```

Runs on [http://localhost:4500](http://localhost:4500).
